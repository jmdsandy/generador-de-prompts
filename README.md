# 🪄 AI Prompt Generator / Generador de Prompts

A powerful, modern web application for creating effective prompts for AI assistants like ChatGPT, Claude, and other language models. Generate custom prompts or choose from a variety of pre-built templates across different categories.

## ✨ Features

### 🎯 Core Functionality
- **Custom Prompt Builder**: Create personalized prompts with role-based personas, specific tasks, context, formatting, and tone
- **Template Library**: 18+ pre-built templates across 6 categories (Writing, Business, Creative, Educational, Coding, Analysis)
- **Interactive Categories**: Easy navigation between different prompt types
- **Real-time Generation**: Instant prompt creation with loading animations

### 💾 Prompt Management
- **Save & Load**: Save your favorite prompts locally for reuse
- **Copy to Clipboard**: One-click copying of generated prompts
- **Prompt History**: View and manage all your saved prompts
- **Delete & Organize**: Remove unwanted prompts from your collection

### 🎨 User Experience
- **Modern UI**: Beautiful gradient design with smooth animations
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Feedback**: Visual feedback for all user actions
- **Keyboard Shortcuts**: Press Enter to generate prompts quickly

### 🛠️ Technical Features
- **Local Storage**: All data persists locally in your browser
- **No Server Required**: Runs entirely in the browser
- **Fast & Lightweight**: Optimized for performance
- **Cross-browser Compatible**: Works in all modern browsers

## 🚀 Getting Started

### Quick Start
1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Start generating prompts immediately!

### No Installation Required
This is a client-side web application that requires no installation or server setup. Simply open the HTML file in your browser and you're ready to go.

## 📖 How to Use

### 1. Using Templates
- Browse through 6 different categories: Writing, Business, Creative, Educational, Coding, and Analysis
- Click on any template card to load it into the custom builder
- Templates provide a starting point with placeholders you can customize

### 2. Custom Prompt Building
- **Role/Persona**: Choose what type of expert the AI should act as
- **Task/Goal**: Describe what you want the AI to do
- **Context**: Provide background information or specific requirements
- **Output Format**: Specify how you want the response formatted
- **Tone/Style**: Set the appropriate tone for your needs

### 3. Managing Prompts
- **Generate**: Click the generate button to create your prompt
- **Copy**: Use the copy button to copy prompts to your clipboard
- **Save**: Save prompts for future use
- **Delete**: Remove saved prompts you no longer need
